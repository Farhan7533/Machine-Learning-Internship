# House Price Predictor - Machine Learning Workflow 🏡

This repository contains the end-to-end Machine Learning pipeline developed as part of my Data Science & AI Internship at **MAINCRAFTS TECHNOLOGY**.

### 📌 Project Objective
To explore the California Housing dataset and build a Linear Regression model to predict house prices based on various demographic and geographical features.

### 🛠️ Tech Stack & Skills Used
* **Language:** Python
* **Data Manipulation & Analysis:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn (Multiple Linear Regression)

### 📊 Key Steps Performed
1. **Data Loading & Preprocessing:** Handled the dataset natively using Scikit-learn.
2. **Exploratory Data Analysis (EDA):** Analyzed feature distributions and plotted correlation heatmaps to identify key predictive variables (e.g., Median Income).
3. **Model Training:** Split the data (80/20) and trained a baseline Linear Regression model.
4. **Evaluation:** Assessed model performance using MAE, RMSE, and $R^2$ metrics.

*Note: This is an introductory assignment demonstrating the complete ML lifecycle.*
